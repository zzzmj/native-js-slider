### 2. 原理

淡入淡出效果主要利用了

`opacity`属性和`transition`属性

让图片淡出，图片的`opacity`属性在一段时间内逐渐从`1`变为`0`

淡入呢，则是图片的`opacity`属性在一段时间内逐渐从`0`变为`1`

这个渐变的过程用transition可以轻松动画



### 3. 功能

点击 pre ，next 切换上一张，下一张图片

鼠标移入下方指示符小圆点，可切换图片
